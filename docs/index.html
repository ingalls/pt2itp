main

Accepts a feature collection of MultiLineStrings - one for each identically named street. It then lines them up in the array so the start and end of each segment are adjacent and where there is no end=>beginning match it splits it into a separate MultiLineString

settings.degTolerance is the max angle between segments that can be matched. Default 45deg. settings.noIntersect disables self intersection checking

main(streets: any, settings: any)
Parameters
streets (any)
settings (any = {})
